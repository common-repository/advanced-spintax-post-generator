== Advanced Spintax Post Generator - Spintax Plugin for WordPress ==

Advanced Spintax Post Generator allows you to quickly create posts using your spintax templates. 

Create multiple locations, events, posts and pages without duplicate content penalties using this free WordPress spintax plugin. For example, if you have multiple stores that all offer the same products, you can quickly create unique pages for each one.

This is a completely free (GPLv2) plugin created by [DesignSmoke - Portland Web Development](https://www.designsmoke.com/)

**Features**

- Avoid Duplicate Content Penalties
- Spin titles, descriptions, slugs, tags and categories
- Supports all custom post types
- Create as many posts as you want

- (Future release) Create templates for easy re-use
- (Future release) Featured images
- (Future release) Create posts for each keyword in a list (EG: Store San Diego, Store Dallas, Store New York City)
